Scanner Progetti Twine

Questo strumento scansiona automaticamente i progetti Twine nella directory "progetti" ed estrae i metadati dai file HTML.

Istruzioni

Per utilizzare lo scanner, segui questi passaggi:

  1. Assicurati che i tuoi progetti Twine siano organizzati in cartelle per classe (es. "progetti/classe1A/")
  2. I file HTML dei progetti devono contenere metadati nei commenti HTML o nei meta tag:
    • Titolo: tag <title>
    • Descrizione: meta tag "description" o primo paragrafo
    • Autori: commento HTML <!-- authors: Nome1, Nome2 -->
    • Tag: commento HTML <!-- tags: tag1, tag2 -->
  3. Clicca sul pulsante "Avvia scansione" per generare automaticamente il file projects.json
